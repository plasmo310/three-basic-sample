# three-basic-sample

* Three.jsの基本的な使い方についてのサンプル集です。

## 各サンプル

* 各フォルダ配下で`npx serve .`を実行することでブラウザ起動して確認できます。

| DirName | Description |
| -- | -- |
| 01_basic_scene | ・Cubeを回転させるだけのシンプルなシーンです。<br><img width="400px;" src="/ReadMeContents/01_basic_scene.png"> |
| 02_basic_scene_improvements | ・`01_basic_scene`をクラス分けやチェック処理追加などして改良したものです。 |
| 03_multi_prim_material | ・標準で用意されているPrimitive、Materialのバリエーションサンプルです。<br><img width="400px;" src="/ReadMeContents/03_multi_prim_material.png"> |
| 04_load_assets | ・FBX、GLTF、Textureの読み込みサンプルです。<br><img width="400px;" src="/ReadMeContents/04_load_assets.png"> |
| 05_custom_shader | ・Shader（トゥーン+アウトライン）、ポストプロセス実装のサンプルです。<br><img width="400px;" src="/ReadMeContents/05_custom_shader_no_post.png"><br><img width="400px;" src="/ReadMeContents/05_custom_shader.png"> |
